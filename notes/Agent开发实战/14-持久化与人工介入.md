# 14 持久化与人工介入

## 学习目标

使用 Checkpointer 保存图状态，通过 `interrupt()` 暂停危险操作，再使用 `Command(resume=...)` 恢复。

## Checkpoint

```python
graph = builder.compile(checkpointer=InMemorySaver())
config = {"configurable": {"thread_id": "refund_1001"}}
```

每次执行都必须使用稳定 `thread_id`。生产部署使用持久化 Checkpointer，并做好租户隔离、加密和过期清理。

## interrupt

```python
from langgraph.types import interrupt


def approval_node(state):
    approved = interrupt({
        "action": "refund",
        "amount": state["amount"],
    })
    return {"approved": bool(approved)}
```

第一次运行到这里会暂停，并把审批信息返回应用。应用把信息展示给有权限的人，而不是让模型审批自己。

## 恢复

```python
from langgraph.types import Command

result = graph.invoke(
    Command(resume=True),
    config=config,
)
```

恢复必须使用同一 `thread_id`。`resume=True` 是外部审批结果，不是模型生成字段。

## 幂等与恢复

Checkpoint 防止丢失工作流状态，但不自动保证业务写操作幂等。退款服务仍需订单号、退款意图 ID、数据库唯一约束和状态查询。
