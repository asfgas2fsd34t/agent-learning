# 重复调用与预算控制速查表

## 调用签名

```python
signature = f"{tool_name}:{json.dumps(arguments, sort_keys=True)}"
call_counts[signature] = call_counts.get(signature, 0) + 1
```

相同工具和相同参数才算同一次重复调用。不同参数的查询不应该被误判为重复。

## 三层保护

```text
max_same_call：限制同一工具和参数
max_steps：限制整个 Agent 任务
timeout / budget：限制时间和费用
```

## 超限返回

```json
{
  "success": false,
  "error_code": "DUPLICATE_TOOL_CALL",
  "message": "相同工具和参数已达到重复调用上限",
  "retryable": false
}
```

保护结果应放回 `messages`，让模型看到失败原因并结束或改变策略；程序仍然要在外层保留最终的步骤上限。
