# 退款状态机速查表

## 状态

```text
PENDING_APPROVAL：等待人工审批
PROCESSING：已获得执行权，正在调用外部支付系统
SUCCEEDED：外部系统确认成功
FAILED：外部系统确认失败或审批拒绝
UNKNOWN：请求超时，暂时不知道外部系统是否成功
```

## 合法转换

```text
PENDING_APPROVAL --APPROVED--> PROCESSING
PENDING_APPROVAL --REJECTED--> FAILED
PROCESSING --PROVIDER_SUCCEEDED--> SUCCEEDED
PROCESSING --PROVIDER_FAILED--> FAILED
PROCESSING --PROVIDER_TIMEOUT--> UNKNOWN
UNKNOWN --RECONCILED_SUCCEEDED--> SUCCEEDED
UNKNOWN --RECONCILED_FAILED--> FAILED
```

## 关键原则

- 超时不等于失败
- UNKNOWN 状态不能直接再次执行写操作
- 通过外部查询或回调对账恢复 UNKNOWN
- SUCCEEDED 和 FAILED 是终态，不能重新执行
- 状态转换必须由程序控制，不能由模型自由决定
