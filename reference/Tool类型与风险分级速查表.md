# Tool 类型与风险分级速查表

## 按业务作用

| 类型 | 作用 | 示例 |
|---|---|---|
| 查询类 | 读取数据，不修改状态 | 查询订单、销售额、库存 |
| 写操作类 | 修改数据或触发动作 | 创建订单、修改地址、保存记忆 |
| 搜索类 | 查找互联网或知识库内容 | 网页搜索、企业知识库搜索 |
| 计算类 | 执行精确计算或代码 | Python、SQL、统计分析 |
| 文件类 | 读取、生成和修改文件 | PDF、Excel、报告 |
| 通信类 | 向外部对象发送消息 | 邮件、短信、企业微信 |
| 系统类 | 操作开发或运行环境 | 日志、测试、Git、文件系统 |

## 按风险分级

```text
低风险：只读查询
中风险：修改数据、发送消息
高风险：退款、付款、删除、权限变更
```

## 控制矩阵

| 控制项 | 查询 | 写操作 | 高风险操作 |
|---|---:|---:|---:|
| 参数校验 | 必须 | 必须 | 必须 |
| 用户和租户权限 | 必须 | 必须 | 必须 |
| 幂等控制 | 通常不需要 | 建议 | 必须 |
| 审计日志 | 建议 | 必须 | 必须 |
| 人工确认 | 通常不需要 | 视业务 | 通常需要 |
| 结果复核 | 建议 | 必须 | 必须 |

## 判断顺序

```text
是否只读？
-> 是否修改真实状态？
-> 是否涉及资金、删除或权限？
-> 确定权限、幂等、审计和人工确认级别
```
