# Agent 开发 Resources

## Knowledge

- [OpenAI: Function calling guide](https://platform.openai.com/docs/guides/function-calling)
  介绍模型如何提出工具调用、应用程序如何执行工具以及如何把工具结果返回模型。用于学习 Tool Calling 的消息结构和责任边界。
- [OpenAI API: Responses API tools](https://platform.openai.com/docs/guides/tools)
  介绍模型使用外部工具的接口设计和工具调用流程。用于后续比较不同 API 风格的工具编排方式。
- [Python: unittest documentation](https://docs.python.org/3/library/unittest.html)
  Python 官方测试框架文档。用于理解单元测试、测试替身和测试发现机制。
- [uv documentation](https://docs.astral.sh/uv/)
  Python 项目和环境管理工具的官方文档。用于维护本项目的 Python 版本、虚拟环境和锁文件。
