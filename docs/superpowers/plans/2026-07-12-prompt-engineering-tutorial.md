# 提示词工程系统教程 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在 `notes` 中创建一份面向开发者、聚焦提示词设计与生产维护的中文系统教程。

**Architecture:** 教程使用单个 Markdown 文件，按“原理 -> 写法 -> 调试 -> 评测与安全”的学习顺序组织。每个核心概念使用问题、反例、改进提示词和总结组成最小学习单元，避免将 RAG、工具调用或 Agent 实现混入范围。

**Tech Stack:** Markdown、UTF-8、现有 `notes` 文档结构。
