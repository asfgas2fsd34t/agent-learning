# Mission: Agent 开发能力

## Why

面向开发者，系统掌握从大模型 API、Tool Calling、Agent 架构、Memory、RAG 到 Agent 工程化的实际开发能力。最终能够独立实现可运行、可测试、可上线的 Agent 应用，而不只是理解概念。

## Success looks like

- 能够从零实现一个调用大模型 API 的 Python 应用
- 能够设计并实现可靠的 Tool Calling 闭环
- 能够为 Agent 添加状态、记忆、权限、重试和人工确认机制
- 能够用单元测试和真实接口测试验证 Agent 行为
- 能够判断一个业务问题是否应该使用 Agent，并说明工程取舍

## Constraints

- 使用中文学习和记录
- 以开发者视角为主，优先结合可运行代码和真实工程问题
- 当前项目使用 Python 3.11 和 OpenAI-compatible API
- 学习过程采用短课、动手练习、回答问题和即时纠正

## Out of scope

- 当前阶段不深入 RAG、MCP 和复杂 Multi-Agent 实现
- 不把 API Key 等敏感配置提交到 Git
- 不用大段理论替代可验证的代码实践
